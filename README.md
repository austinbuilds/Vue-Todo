# :clipboard: Vue Todo

Vue to-do list application demonstrating and understanding of Vue including setup using Vue CLI, components, data, events, directives, API integration, and routing.

## :floppy_disk: Install

Dependencies:

- @vue/cli
- vue-router
- uuid
- axios

Recommeneded:

- @vue/cli

Run:

- npm run serve
